Termo Significado
A
API API significa Application Programming Interface (Interface de Programação de Aplicação). No contexto de APIs, a palavra Aplicação refere-se a qualquer software com uma função distinta. A interface pode ser pensada como um contrato de serviço entre duas aplicações.
Arduíno O nome Arduino vem de um bar na comuna italiana de Ivrea, onde alguns dos fundadores do projeto costumavam se reunir. O bar foi nomeado após Arduíno de Ivrea, que foi o marquês de Ivrea e rei da Itália de 1002 à 1014.
B
Back-End As linguagens de programação backend lidam com os bastidores das funcionalidades das aplicações web. É o código que conecta a internet com o banco de dados, gerencia as conexões dos usuários e alimenta a aplicação web. O backend trabalha em conjunto com o frontend para entregar o produto para o usuário final.
BIOS BIOS é a sigla para Basic Input/Output System, ou Sistema Integrado de Entrada e Saída. Por muito tempo, foi o programa responsável por iniciar os sistemas operacionais e as tarefas mais importantes de um computador, a partir do momento em que ele é ligado.
Bug É a falha, ou erro em um programa computacional, que o impede de se comportar como pretendido. A maioria dos erros são causados por compiladores produzindo um código inválido.
C
Cache O cache é, de modo geral, o lugar onde os arquivos temporários do seu smartphone (e dos computadores também) ficam armazenados para que possam ser acessados mais rapidamente quando necessário, agilizando a execução dos programas.
Cookie Cookies são arquivos de texto que um site envia ao navegador de um usuário, com dados sobre o seu perfil e comportamento de navegação nas páginas. O objetivo dos cookies é lembrar das informações da visita para facilitar o próximo acesso e melhorar sua navegação no site.
CSS CSS é a sigla para Cascading Style Sheets, ou seja, Folhas de Estilo em Cascatas. É uma maneira de dar estilo ao código criado por linguagens como HTML, XML ou XHTML, por exemplo.
D
Datacenter ou CPD O Centro de Processamento de Dados (CPD), são locais físicos que ficam dentro das empresas. Neste ambiente são concentrados todos os servidores e serviços de tecnologia, importantes para o gerenciamento da informação.
Dashboard Um dashboard é um painel de informações que contém métricas e indicadores-chave de performance. Esse painel geralmente é monitorado por toda a equipe. Serve para embasar decisões e acompanhar o desempenho da empresa.
DNS (Domain Name System - Sistema de Nomes de Domínio) É um sistema que traduz nomes de domínio (exemplo.com) em endereços IP para facilitar a navegação na internet.
E
Ethernet É um padrão de tecnologia de rede com fio que define como os dispositivos se comunicam em uma rede local.
F
Framework É um conjunto de ferramentas, bibliotecas e diretrizes que ajudam os desenvolvedores a criar aplicativos de software mais facilmente.
Front-End É a parte de um site ou aplicativo que os usuários veem e interagem, responsável pela interface do usuário.
FTP (File Transfer Protocol - Protocolo de Tranferência de Arquivos) É um protocolo de transferência de arquivos que permite o envio e o recebimento de arquivos pela internet.
G
Gateway É um dispositivo ou software que conecta duas redes diferentes, permitindo a comunicação entre elas.
H
Hardware São os componentes físicos de um computador ou dispositivo eletrônico, como processador, memória, teclado, monitor, etc.
Hotspot É um local onde é possível acessar a internet sem fio, geralmente por meio de Wi-Fi, fornecido por um roteador ou dispositivo sem fio.
HTML (Hypertext Markup Language - Linguagem de Marcação de Hipertexto) É uma linguagem de marcação usada para criar páginas da web, define a estrutura e o conteúdo das páginas web como esta.
HTTP ou HTTPS (Hypertext Transfer Protocol / Secure - Protocolo de Transferência de Hipertexto / Seguro) São protocolos de transferência de hipertexto usados para a comunicação entre um navegador e um servidor web. O HTTPS é a versão segura, que utiliza criptografia.
Hub É um dispositivo de rede que atua como um ponto central para conectar vários dispositivos em uma rede local, mas geralmente é menos inteligente do que um switch.
I
IOT (Internet of Things - Internet das Coisas) Refere-se a objetos e dispositivos que podem se comunicar pela Internet e trocar dados entre si, permitindo automação e controle remoto. Os exemplos incluem termostatos inteligentes e iluminação controlada por aplicativos.
IP (Internet Protocol - Protocolo de internet) É o protocolo que permite que dispositivos se comuniquem pela internet. Ele atribui endereços exclusivos aos dispositivos para que possam enviar e receber informações uns dos outros.
J
Java Java é uma linguagem de programação amplamente utilizada, conhecida por sua portabilidade e capacidade de execução em diferentes sistemas. É comumente usado no desenvolvimento de aplicativos web e móveis.
JavaScript JavaScript é uma linguagem de programação usada principalmente para interatividade em sites. Funciona em um navegador da web e permite criar páginas da web dinâmicas e responsivas.
K
L
LAN (Local Area Network - Rede Local de Internet) Uma rede de computadores que cobre uma área geográfica limitada, como uma casa, um escritório ou um edifício. Permite a comunicação e o compartilhamento de recursos entre dispositivos próximos.
M
MAN (Metropolitan Area Network - Rede Metropolitana de Internet) Uma rede que cobre uma cidade, como uma cidade. Ele conecta várias redes locais (LANs) em uma grande área geográfica.
N
NFC (Near Field Communication - Comunicação de Campo Próximo) É uma tecnologia que permite a comunicação perfeita entre dispositivos, geralmente a poucos centímetros de distância. É comumente usado para pagamentos móveis e transferência de dados entre smartphones.
O
P
PHP (Hypertext Preprocessor - Pré-processador de Hipertexto) PHP é a linguagem de programação mais usada para desenvolvimento web. É comumente usado em servidores web para criar páginas dinâmicas e interativas.
Q
Query - (Consulta) Na computação, uma query é uma consulta ou solicitação de informações inseridas em um banco de dados. Usado para recuperar dados específicos de um banco de dados. SGBD (Sistema Gerenciador de Banco de Dados)**: É um software que permite armazenar, organizar e recuperar informações em um banco de dados. Ele fornece ferramentas para criar, transformar e consultar dados com eficiência.
R
S
Software É uma serie de comandos escritos interpretados por um computador para executar tarefas específicas.
SQL (Structured Query Language - Linguagem de Consulta Estruturada) É uma linguagem padrão de gerenciação de banco de dados relacionais. SQL ndo Linguagem de Consulta Estruturada.
Switch É um dispositivo de concentração para conetar diversos computadores em uma conexão de rede local.
T
TCP/IP (Transmission Control Protocol - Protocolo de Controle de Transmissão) É um conjunto de protocolos para comunicação entre computadore e servidores, formando a internet.
U
URL (Uniform Resource Locator) É o localizador Uniforme de Recursos o endereço virtual de uma pagina ou website.
UX (User Experience) Significa experiência de usuario.
V
VM (Virtual Machine - Maquina virtual) É um ambiente de uso criado por software usado para rodar sistemas operacionais diferentes em um computador ao mesmo tempo.
VPN (Virtual Private Network - Rede Privada Virtual) Pode ser usado para estabelecer uma conexão de rede protegida ao usar redes públicas
W
Wan (Wide Area Network - rede de longa distância) É uma rede de computadores que abrange uma grande área geográfica como um país.
Web Traduzindo literalmente Teia, mais usado para referir-se a internet.
X
XML (Extensible Markup Language - Linguagem de Marcação Extensível) Muito usado para contruções de telas de aplicativos androids.
Y
Z
